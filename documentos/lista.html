Realizando un ramo

  1. Tener una docena de rosas.
  2. Tener un rollito de follaje y nuvecita.
  3. Dejar absorber agua un oasis.
  4. Todas las rosas se deberan limpiar desde las hojas hasta los petalos.
  5. Ya teniendo listas las rosas se cortara el tallo a una medida deseda.
  6. Ya que el oasis absorbio el agua se iniciara colocar el follaje y la nuvecita (no llenar todo)
  7. Ya teniendo todo listo se acomodara la rosa iniciando con una cruz y lo que falte se termina de rellenar
  8. ¡Y listo tienes una decoración sencilla y bonita!
